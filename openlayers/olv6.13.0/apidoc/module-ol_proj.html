module:ol/proj.addCoordinateTransforms(source, destination, forward, inverse)

import {addCoordinateTransforms} from 'ol/proj';

Registers coordinate transform functions to convert coordinates between the source projection and the destination projection. The forward and inverse functions convert coordinate pairs; this function converts these into the functions used internally which also handle extents and coordinate arrays.

Name Type Description

destination module:ol/proj~ProjectionLike

Destination projection.
forward function

The forward transform function (that is, from the source projection to the destination projection) that takes a module:ol/coordinate~Coordinate as argument and returns the transformed module:ol/coordinate~Coordinate.
inverse function

The inverse transform function (that is, from the destination projection to the source projection) that takes a module:ol/coordinate~Coordinate as argument and returns the transformed module:ol/coordinate~Coordinate.

module:ol/proj.getPointResolution(projection, resolution, point, opt_units){number}

proj.js, line 200 
import {getPointResolution} from 'ol/proj';

Get the resolution of the point in degrees or distance units. For projections with degrees as the unit this will simply return the provided resolution. For other projections the point resolution is by default estimated by transforming the point pixel to EPSG:4326, measuring its width and height on the normal sphere, and taking the average of the width and height. A custom function can be provided for a specific projection, either by setting the getPointResolution option in the module:ol/proj/Projection~Projection constructor or by using module:ol/proj/Projection~Projection#setGetPointResolution to change an existing projection object.

Name Type Description
projection module:ol/proj~ProjectionLike

The projection.
resolution number

Nominal resolution in projection units.
point module:ol/coordinate~Coordinate

Point to find adjusted resolution at.
units module:ol/proj/Units | undefined

Units to get the point resolution in. Default is the projection's units.
Returns:
Point resolution.
